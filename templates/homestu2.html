SIDEBAR LINK DESCRIPTIONS

Create User

The "Create User" link allows you to add new users to the system.
You can fill in details such as username, password, role, phone number, and email. This section is useful for managing user accounts and roles.

Operator Details

The "Operator Details" link provides access to the information and details of operators within the system.
This may include their personal details, roles, and other relevant information.

Fetch Students Internship Data

The "Fetch Students Internship Data" link allows you to retrieve and view internship data for students within a specified date range.
You can select start and end dates to filter the data accordingly.

Mentor Acceptance Forms

The "Mentor Acceptance Forms" link enables you to fetch and view specific mentor acceptance forms based on a selected date.
This allows you to manage and review mentor engagements for specific time periods.

Certificates Not Collected

The "Certificates Not Collected" link displays a list of students who have not yet collected their certificates.
This section helps in tracking and managing certificate distribution.

Internship Verification

The "Internship Verification" link is used to verify if the student has done the internship at ECIL or not.
